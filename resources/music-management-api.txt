MUSIC MANAGEMENT APP - REST API
===============================

Artist
	- GET   - /artist                   -->   + getAllArtists()                         : List<ArtistDTO>
	- GET   - /artist/:id               -->   + getArtistById(id: Long)                 : ArtistDTO
	- POST  - /artist/add               -->   + insertArtist(artistForm: ArtistForm)    : ArtistDTO
	- PUT   - /artist/:id               -->   + updateArtistById(id: Long, name: String): ArtistDTO
			

Genre
	- GET   - /genre                    -->   + getAllGenres()                         : List<GenreDTO>
	- GET   - /genre/:id                -->   + getGenreById(id: Long)                 : GenreDTO
	- POST  - /genre/add                -->   + insertGenre(genreForm: GenreForm)      : GenreDTO
	- PUT   - /genre/:id                -->   + updateGenreById(id: Long, name: String): GenreDTO
	
	
Album
	- GET   - /album/artist/:artistId   -->   + getAlbumsByArtist(artistId: Long)              : List<AlbumDTO>
	- GET   - /album/:id                -->   + getAlbumById(id: Long)                         : AlbumDTO
	- POST  - /album/add                -->   + insertAlbum(albumForm: AlbumForm)              : AlbumDTO
	- PUT   - /album/:id                -->   + updateAlbumById(id: Long, albumForm: AlbumForm): AlbumDTO
	

Track
	- GET   - /track/artist/:artistId   -->   + getTracksByArtist(artistId: Long)              : List<TrackDTO>
	- GET   - /track/album/:albumId     -->   + getTracksByAlbum(albumId: Long)                : List<TrackDTO>
	- GET   - /track/:id                -->   + getTrackById(id: Long)                         : TrackDTO
	- POST  - /track/add                -->   + insertTrack(trackForm: TrackForm)              : TrackDTO
	- PUT   - /track/:id                -->   + updateTrackById(id: Long, trackForm: TrackForm): TrackDTO
